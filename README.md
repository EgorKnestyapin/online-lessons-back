## Project setup

* It is necessary to ensure the presence of Environment Variables
  * For database:
    * `DB_HOST` 
    * `DB_PORT` 
    * `DB_NAME` 
    * `DB_USERNAME` 
    * `DB_PASSWORD`
  * For working with Digital Ocean/AWS S3
    * `s3_accessKey`
    * `s3_endpoint`
    * `s3_region`
    * `s3_secretKey`
